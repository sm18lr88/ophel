# Privacy Policy

**Last Updated**: 2026-03-08

## Overview

Ophel is a browser extension and userscript that enhances AI chat interfaces. It does not operate a hosted backend for user data.

## Data Collection

Ophel does not collect, transmit, or store your data on OpenAI-controlled or project-owned servers.

Data is stored locally in your browser unless you explicitly configure your own sync destination.

Examples of locally stored data:

- Settings and preferences
- Prompt templates
- Conversation organization metadata such as folders and tags
- Reading-history positions
- Theme selections

## Optional Sync

If you enable WebDAV sync, your data is sent to the WebDAV server you configure. That server is under your control, not ours.

## Permissions

Ophel requests only the permissions needed for its features.

- `storage`: persist settings and local data
- site host permissions: run on supported AI sites
- `notifications` (optional): completion alerts
- `cookies` (optional): Claude session-key management

## Security and Privacy

- Data remains local by default
- No analytics
- No ad network integrations
- No remote profiling or tracking
