# 核心功能

Ophel 的核心工作区由三个主功能页组成：`智能大纲`、`会话管理`、`提示词助手`。这三个模块负责日常使用中的结构化阅读、会话组织与提示词复用。

## 三大核心模块

| 模块       | 解决问题                       | 入口                       |
| ---------- | ------------------------------ | -------------------------- |
| 智能大纲   | 长对话快速定位与结构导航       | [查看详情](./outline)      |
| 会话管理   | 会话分类、检索、批量处理与导出 | [查看详情](./conversation) |
| 提示词助手 | 模板化 Prompt 复用与变量化输入 | [查看详情](./prompt)       |

## 与设置中心的关系

核心模块的行为由“设置中心”统一配置，例如：

- 大纲自动更新、跟随模式、收藏图标显示
- 会话同步策略、导出命名规则、图片导出策略
- Prompt 双击发送、发送键位策略
- 标签页自动重命名、隐私模式、完成通知
- 内容交互增强（用户问题 Markdown、LaTeX/表格复制）

可以在 [设置总览（体验增强）](/zh/guide/enhancements) 中查看完整菜单与路径。

## 推荐阅读顺序

1. 先看 [智能大纲](./outline)：建立长对话导航能力。
2. 再看 [会话管理](./conversation)：完成会话组织与导出策略。
3. 最后看 [提示词助手](./prompt)：沉淀可复用 Prompt 资产。
